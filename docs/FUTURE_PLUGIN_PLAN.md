# Future Plugin Plan

## Planned Plugins

### ConsoleCore Sender
Placed on DAW channels/groups.
Responsibilities:
- send audio into ConsoleCore ecosystem
- register stem identity
- participate in shared rail state

### ConsoleCore Return
Placed on mix bus.
Responsibilities:
- receive processed summing return
- expose console macros
- monitor stress/rail behavior

### ConsoleCore Control
Global management:
- topology visualization
- routing
- console selection
- automation snapshots
