# Milestone D Specification

## Goal

Build the framework that allows ConsoleCore to evolve from generic analog physics into measured, profile-driven console behavior.

## Major Systems

1. Capture Session Format
2. Test Signal Generator
3. Measurement Importer
4. Feature Extraction
5. Model Fitting
6. Profile Validator
7. Generic Console Model Library

## Supported Measurement Types

- Sine sweep
- Multitone
- Impulse
- Step response
- Pink noise
- Burst/transient stress
- Stereo crosstalk probe
- Level-dependent harmonic response
- Dynamic recovery/sag response

## Legal Boundary

ConsoleCore should support measured profiles named by the user, but built-in public profiles must use generic family names unless a license or explicit authorization exists.
