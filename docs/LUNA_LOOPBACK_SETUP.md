# LUNA + Loopback Recommended Workflow

1. Create Loopback virtual output:
   ConsoleCore Sends

2. Create Loopback return:
   ConsoleCore Returns

3. Route LUNA buses/stems:
   Drums
   Bass
   Guitars
   Vocals
   FX
   Parallel
   etc.

4. ConsoleCore receives:
   multichannel stem input

5. ConsoleCore processes:
   TapeCore Track
   Console Physics
   Shared Rail
   Print Tape

6. Return stereo mix or stems to:
   ConsoleCore Returns
