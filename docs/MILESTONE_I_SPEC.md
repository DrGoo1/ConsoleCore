# Milestone I Specification

## Goal

Prepare ConsoleCore for:
- public beta
- commercial distribution
- cross-DAW deployment
- stable user support

## Major Systems

1. Licensing abstraction
2. Preset/package manager
3. Crash reporting
4. Diagnostics bundle generation
5. User-profile persistence
6. Session portability
7. Installer/update planning
8. Beta telemetry
9. Release configuration management

## Commercial Targets

- Standalone macOS app
- AU plugin suite
- VST3 plugin suite
- Future Windows support
