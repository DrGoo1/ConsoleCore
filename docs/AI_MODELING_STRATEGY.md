# AI Modeling Strategy

ConsoleCore should remain fundamentally:
- deterministic
- measurable
- engineerable

AI should augment:
- profile discovery
- matching
- clustering
- parameter estimation
- behavior classification

AI should NOT replace:
- core DSP integrity
- realtime stability
- explainable console behavior

Recommended architecture:
DSP core remains explicit.
AI layers provide:
- parameter recommendations
- profile interpolation
- similarity ranking
- adaptive routing suggestions
