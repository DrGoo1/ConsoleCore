# Spatial Architecture

ConsoleCore spatial processing should model:
- shared rail interaction across immersive buses
- inter-speaker coupling
- dynamic spatial energy distribution
- multichannel transformer/tape stress
- topology-aware immersive summing

This should NOT simply duplicate stereo processing across channels.

Large LFE bursts should:
- influence rail stress
- affect headroom
- interact with front channels

similar to real analog infrastructure.
