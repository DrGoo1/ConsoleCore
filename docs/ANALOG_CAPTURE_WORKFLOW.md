# Analog Capture Workflow

## Recommended Workflow

DAW Test Generator
↓
Converter Output
↓
Analog Hardware Chain
↓
Converter Input
↓
ConsoleCore Capture Session
↓
Feature Extraction + Profile Fit

## Required Documentation

For every capture:
- sample rate
- bit depth
- converter model
- gain staging
- routing
- cable path
- hardware settings
- calibration tone reference

## Recommended Capture Types

1. Clean calibration pass
2. Multi-level sweeps
3. Dynamic transient passes
4. Crosstalk characterization
5. Rail stress tests
6. Noise-floor captures
7. Frequency-dependent saturation
